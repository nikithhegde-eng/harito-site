# Vendor Catalog Onboarding

Use this process when vendors reply with product data or partnership terms.

## Data To Request

- SKU
- Product title
- Brand
- Vendor name
- Category
- Product type
- Gender
- Colour
- Size list
- MRP
- Selling price
- Stock quantity or stock status
- Product images
- Product description
- HSN/GST/tax details
- Return eligibility
- Delivery ownership
- Commission or margin
- Vendor payout timing
- Usage rules for images, logos, and product copy

## Import Rule

Do not add vendor products to the public catalog until written approval is stored in `VENDOR_TRACKER.md`.

## Template

Send vendors `vendor-catalog-template.csv` and ask them to return the completed file or an equivalent feed.
