# Harito Project Progress

## What's built
- server.js: Express backend with orders API, 20% commission tracking
- cart.html: Shop page with product catalog and Buy Now flow  
- admin.html: Admin dashboard with commission tracking
- harito.in: Live on Netlify

## What's next
- [ ] Razorpay payment integration (need keys from razorpay.com)
- [ ] Connect Netlify to GitHub for auto-deploy
- [ ] Add real vendor products to catalog
- [ ] Buyer email confirmation after order

## Stack
- Frontend: HTML, CSS, vanilla JS
- Backend: Node.js + Express
- Hosting: Netlify (harito.in)
- Repo: https://github.com/nikithhegde-eng/harito-site

## How to run locally
npm install express
npm start
open http://localhost:4173
